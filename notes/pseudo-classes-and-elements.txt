A pseudo-class is a selector that selects elements that are in a specific state,
for example, they are the first element of their type, or they are being hovered
over by the mouse pointer.

Pseudo-classes are keywords that start with a colon. For example, :hover is
a pseudo-class.

A pseudo-element behaves similarly to a pseudo-class except that act as if you
had added a whole new HTML element into the markup, rather than apply a class to
existing elements.

Pseudo-elements start with a double colon '::'
- e.g. ::before

Pseudo-classes and elements can be combined

There are a couple of secial pseudo-elements, which are used along with the
content property to insert content into your document using CSS. This technique
is called *generated content*.
